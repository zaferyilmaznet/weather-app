# Local Weather App

A simple **weather app** that displays current weather information for a city. Built as a **training project** using HTML, CSS, and JavaScript, with a **serverless proxy** to securely hide the OpenWeatherMap API key.

## Features

- Mobile-first responsive design
- Light and dark mode support
- Weather details including:
  - Temperature
  - Real feel
  - Wind
  - Chance of rain
  - Pressure
- Emoji-enhanced UI for clarity
- Serverless API proxy (Vercel) to keep API key secure

## Tech

- HTML, CSS, Vanilla JS
- OpenWeatherMap API
- Serverless proxy (Vercel Functions / Netlify Functions)
- Deployed on Vercel

## Deployment

This project is deployed on Vercel using Serverless Functions.
API keys are securely stored using environment variables and are never exposed to the client.
